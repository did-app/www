Features Grid

One Click Sign In

Users sign in with a single click using a private key stored on their device.

Very Very Fast

Device authentication is the fastest way to create an account or sign in.

Quick Development

Install with just five lines of code and works with open standards: OAuth and Open ID Connect.

Email Fallback

DID uses email authentication when the user signs in from an unrecognised device.

Set and Forget

Install once and forget all about it. We maintain your authentication so you don't have to.

Active Device Management

Users can view a list of all their trusted devices and selectively forget them. Devices are automatically untrusted if reset.

QR Scan Sign In

Users signing in on input-less or shared devices can scan a QR code with their trusted smartphone for instant authentication.

Open Source

DID's key and signature generation is open source for transparency. View the source code here.

DID's User Network

Previously authenticated DID users will be able to sign up to your website in just one click.

Signing in with DID

DID offers two passwordless ways for users to sign in. They can either authenticate using their email address or their device. No app required.

Users already on DID can create an account with a single click. Reducing friction, and improving conversion.
